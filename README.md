## Featured Projects

### [Midea AC IR Protocol Decode](https://github.com/smruchira/Midea-AC-IR-Protocol-Decode)

Reverse-engineered the infrared protocol used by a Midea air conditioner and RG57A7/BGEF remote. The project documents pulse timings, 48-bit packet structure, checksum behaviour, non-linear temperature encoding, special command packets, and an ESP32/Arduino implementation.

**Keywords:** reverse engineering, embedded C++, ESP32, Arduino, infrared protocol, checksum analysis, IoT

### [Popylabs ERP](https://github.com/smruchira/Popylabs-ERP)

A multi-tenant SaaS ERP platform designed for Sri Lankan SMEs. It supports organisation-scoped data, products, stock, sales, atomic transactions, invoice generation, transaction history, offline persistence, PWA functionality, and secure Firebase access rules.

**Technology:** Next.js, TypeScript, Tailwind CSS, Firebase Authentication, Firestore, Vercel, jsPDF

### [Popylabs Website](https://github.com/smruchira/Popylabs-Website)

The public website for Popylabs, built as a responsive multi-page web application with service pages, interactive components, structured metadata, semantic HTML, and deployment optimisation.

**Technology:** Next.js, React, TypeScript, Tailwind CSS, Vercel

### [Insider-Threat Detection MVP](https://github.com/smruchira/insider_mvp_project_updated_v5)

An academic Python prototype that combines machine learning and optional LLM analysis to identify potentially risky employee online activity, including excessive social-media use and concerning public content. The project uses manual ingestion and is designed as an educational proof of concept rather than a production monitoring platform.

**Keywords:** insider-threat detection, Python automation, machine learning, LLM analysis, risk scoring, Flask
